# Mini-Zepto

## Overview

Mini-Zepto is a lightweight e-commerce project that implements the core functionalities of Zepto, a fast and efficient delivery platform. Built with **Java**, this project demonstrates essential e-commerce features in a simplified manner.

## Project Description

This is a normal e-commerce project which is a mini version of Zepto. It provides a foundation for understanding how e-commerce platforms handle product management, user interactions, and order processing.
